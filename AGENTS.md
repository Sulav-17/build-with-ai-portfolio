# Codex Instructions

## Project Scope

This repository is a personal portfolio built with Next.js App Router, TypeScript, and Tailwind CSS.

## Working Rules

- Preserve a professional, minimal technical design.
- Use semantic HTML and accessible default browser behavior.
- Prefer server-rendered components and normal Next.js links unless client interactivity is explicitly required.
- Do not fabricate personal, professional, project, employment, education, or credential information.
- Do not add backend services, databases, CMS integrations, analytics, authentication, deployment configuration, or APIs unless explicitly requested.
- Do not add packages unless they are necessary and approved.
- Keep styling restrained and avoid animations unless explicitly requested.
- Keep changes scoped to the requested milestone or task.
- Do not commit changes unless explicitly asked.

## Content Rules

- Placeholder pages may describe structure or intent only.
- Real project content should be added only when supplied by the user.
- Project cards and detailed case studies should wait until requested by a later milestone.
